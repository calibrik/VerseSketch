# About
VerseSketch is a web based party game. Players input a few lines of their favorite songs and then take turns to draw association images for each 2 lines from them. In the end it plays a compilation of player drawn images and voiced over lyrics.

(Can be played with yourslef in 2 separate tabs)

# Build locally using docker
1. You need to install Docker.

2. In the project folder run ```docker compose up --build``` in terminal or click on [dockerComposeLocal.bat](dockerComposeLocal.bat) to build images and run containers. Server will start on [localhost:80](http://localhost:80).
